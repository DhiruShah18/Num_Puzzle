ABOUT THIS DEMO

8-Puzzle is a classic AI search problem. The goal is to slide tiles on a 3×3 grid until they match the goal state 1 2 3 / 4 5 6 / 7 8 _.

Manhattan Distance (h)
The heuristic estimates cost by summing, for each tile, the horizontal + vertical steps needed to reach its goal position. It never overestimates → admissible.

A* Search
Expands the node with the lowest f(n) = g(n) + h(n), where g(n) = moves so far. Finds the optimal (shortest) solution.

Greedy Best-First Search
Expands the node with the lowest h(n) only. Faster but not guaranteed to find the optimal path.
